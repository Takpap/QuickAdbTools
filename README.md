# QuickAdbTools
Android 开发个人常用 adb 命令集成 
## 为何要自己造轮子?
#### 一直心心念念想着开发自己的 PC 端小工具, 遂网上学习了几天 WPF .Netcore 开发 PC 端桌面程序, 入门挺简单, 进阶留着以后有时间慢慢来吧, 有了这个工具节省了平时敲 adb 命令的时间, 也为以后折腾 WPF 开发入了一个门
## 主要功能
### 1. 一键盘连接 无线adb 调试
### 2. 本地存储每次连接的 Android 手机的 Ip + Port
### 3. 一键开启 scrcpy 无线投屏
### 4. 拖拽 apk 一键安装
### 5. 拖拽文件一键传输到手机根目录
### 6. 一键开启 adb 全局代理 OR 关闭
